Seamless WooCommerce to WhatsApp Integration by ToolBridges. Send automated order notifications directly to your customers on WhatsApp.

== Description ==
ToolBridges Order Notification enables seamless integration between WooCommerce and WhatsApp, allowing businesses to enhance their customer service by sending automated order notifications directly to WhatsApp. This plugin relies on ToolBridges' third-party API service to ensure reliable and timely message delivery. By using this plugin, you agree to the terms and policies of the service provider.

== Installation ==
1. Upload the `toolbridges-order-notification` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to "Order Notifications" under "Settings" in the admin menu to configure the plugin.
4. Obtain an API key and other necessary details from your ToolBridges account to configure the plugin properly.

== Third-Party Service Information ==
This plugin connects to a third-party API provided by ToolBridges to send WhatsApp notifications. Your use of this plugin and its associated service is subject to ToolBridges' Terms of Service: https://toolbridges.com/terms-and-conditions and Privacy Policy : https://toolbridges.com/privacy-policy

Service URL
The plugin interacts with the following service endpoint: https://app.toolbridges.com/api/wpbox/sendtemplatemessage

== Frequently Asked Questions ==
= How do I get the API key? =
